# Garden AR

An AR Plant Placement App built using Unity allows users to place virtual plants in real-world space by simply tapping on the screen. It helps users visualize how plants will look inside their home, office, or garden using augmented reality.

## Features

* Tap on screen to place AR plants
* Real-time surface detection using AR
* Random plant selection from multiple 3D models
* Accurate placement on detected planes
* Simple and interactive user experience

## Technologies Used

* Unity Engine
* AR Foundation
* ARCore
* C#
* 3D Plant Models

## How It Works

The app detects flat surfaces like floors or tables using the mobile camera. When the user taps on the screen, a virtual plant is placed at that location in augmented reality.

## How to Run

1. Clone the repository
2. Open the project in Unity
3. Install AR Foundation and ARCore/ARKit packages
4. Build and run on an AR-supported Android or iOS device
